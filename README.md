# Анализ кредитного скоринга клиентов

Данный проект посвящён построению модели кредитного скоринга для оценки вероятности дефолта заемщиков. В работе проводится исследование факторов, влияющих на кредитоспособность клиентов, а также построение и интерпретация модели классификации.

---

## 📌 Описание задачи

Основная цель проекта — определить, какие характеристики клиента наиболее существенно влияют на риск невыплаты кредита.

Ключевой вопрос исследования:
**«Можно ли на основе социально-демографических и финансовых показателей предсказать вероятность дефолта заемщика?»**

В рамках проекта:

* анализируются финансовые и поведенческие характеристики клиентов;
* выявляются значимые признаки;
* строится модель бинарной классификации для прогнозирования дефолта.

---

## 📊 Данные

Используется датасет кредитного скоринга, содержащий информацию о клиентах банка.

Основные переменные могут включать:

* `Age` — возраст клиента
* `Income` — уровень дохода
* `Loan_Amount` — сумма кредита
* `Credit_History` — кредитная история
* `Employment_Status` — статус занятости
* `Debt_To_Income` — отношение долга к доходу
* `Number_of_Dependents` — количество иждивенцев

**Целевая переменная:**

* `Default` / `Loan_Status` — факт дефолта (1 — дефолт, 0 — нет)

---

## 🛠 Методология

### 1. Предобработка данных

* Проверка и обработка пропусков
* Кодирование категориальных признаков
* Масштабирование числовых переменных
* Анализ выбросов

### 2. Разведочный анализ (EDA)

* Анализ распределений признаков
* Исследование корреляций
* Сравнение характеристик клиентов с дефолтом и без
* Визуализация зависимостей между признаками

### 3. Моделирование

* Разделение выборки на обучающую и тестовую
* Построение модели логистической регрессии
* Использование деревьев решений и градиентного бустинга
* Подбор гиперпараметров

### 4. Оценка качества модели

* ROC-AUC
* Анализ матрицы ошибок

### 5. Интерпретация результатов

* Анализ значимости коэффициентов
* Определение наиболее важных факторов риска
* Выводы о практической применимости модели

---

## 🚀 Основные выводы

* Финансовые показатели (уровень дохода, долговая нагрузка) оказывают существенное влияние на вероятность дефолта.
* Кредитная история является одним из наиболее значимых предикторов.
* Модель логистической регрессии демонстрирует интерпретируемость и приемлемое качество классификации.
* Использование дополнительных алгоритмов может повысить точность прогноза, но снижает интерпретируемость.
